### 磁盘容量/使用量查询
```shell
df -h
```

### 查询文件夹存储占用细节
```shell
du -h --max-depth=1
```

### 查询磁盘当前传输速度
[参考资料](https://www.cnblogs.com/machangwei-8/p/10388693.html)
```shell
# iostat 每秒钟的cpu和磁盘传输情况，以M为单位，查看10次
iostat -c -d -m 1 10
# iotop
sudo iotop

```
